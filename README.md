## インストール法、始め方

### Gitからクローンしましょう
```
git clone https://github.com/Fel615/IoTDojo_fri2nd
```
### ディレクトリを移動します
```
cd IoTDojo_fri2nd/flask
```
### 仮想環境を作成します
```
python -m venv env
```
### 仮想環境を立ち上げましょう
```
env\Script\actvate
```
### モジュールをインストールします
```
pip install -r requirements.txt
```
### 実行します。
```
python app.py
```
